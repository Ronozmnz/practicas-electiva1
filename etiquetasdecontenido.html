ETIQUETAS DE CONTENIDO

Etiqueta p

Este es un texto dentro de una etiqueta de párrafo. Es ideal para organizar el contenido escrito de tu página web de forma legible.

Cada vez que abres y cierras esta etiqueta, el navegador entiende que comienza un bloque de texto nuevo.

Contenido pre formatiado                   ya que conserva los espacios, 
            o los saltos de linea y los tabuladores del texto original.

Etiqueta blockquote

Con blockquote representamos un contenido citado desde otra fuente.
