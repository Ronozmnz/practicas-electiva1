# Prácticas de Electiva I (2026) 🎓

Este repositorio contiene la recopilación de prácticas y ejercicios desarrollados para la materia de **Electiva I** durante el año 2026. El objetivo principal es dominar las bases del desarrollo web interactivo y el diseño responsivo.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructuración semántica del contenido.
* **CSS3:** Estilos, diseño responsivo, Flexbox y CSS Grid.
* **JavaScript:** Lógica de programación e interactividad del lado del cliente.

## 📂 Contenido del Proyecto

El archivo `index.html` funciona como el menú principal del portafolio, enlazando de forma ordenada cada uno de los ejercicios académicos realizados:
* Estructuras y encabezados
* Manejo de tablas y listas
* Formularios de registro de datos
* Maquetación avanzada (Flexbox & Grid Area)
* Evaluaciones y exámenes prácticos
